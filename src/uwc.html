Universal web chat client, combining the technologies of socket.io and Google Translate.

User Experience and Development

Rochester, NY Winter 2015

    UWC is an in-development web app that allows users who speak different languages to communicate via a simple chat client that translates the senderâ€™s message into the selected language of the receiving user. Using socket.io to create a simple web chat client, outgoing messages get fed through a Google Translate API with the translated message getting piped back into our client.

    The goal of this project is to close the language barrier in online conversation. In early versions the chat client will be an open chat room, inviting international users to chat with one another. In later versions we hope UWC will be able to be used as a one-to-one communication tool, with the ability to invite users by email to a private chat session.

Working with UX designer Alberto Camacho.
